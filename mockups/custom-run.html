TriviO — Custom run builder

Two pieces of beta feedback point at the same feature: "let me pick the topics I want (philosophy, geography…)" and "let me start with lower-difficulty questions." A Custom run screen unifies both — check/uncheck categories, pick difficulty tiers, choose length, hit Start. It also fixes the discoverability gap (the user thought topic selection lived under Profile but "wasn't clear how").

Train tab — a clear Custom run card sits beside the existing modes, so building a filtered set is one obvious tap (no more hunting in Profile).
